# Interglobe LLC website

Static website prepared for deployment at `interglobeus.com` / GitHub Pages.

## Main pages

- `index.html` - home page, capability statement, two core service practices, and Why Interglobe
- `financial-consulting.html` - complete financial consulting service content
- `information-technology.html` - complete information technology, cybersecurity, and risk service content
- `contact-us.html` - contact information and consultation links

## Deployment

Upload the contents of this folder to the root of the GitHub Pages repository. The existing `CNAME` file points to `interglobeus.com`.
